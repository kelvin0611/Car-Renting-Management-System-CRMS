1. 成為 Owner

註冊並完成實名驗證，在 Owner Dashboard 新增車輛資料與相片。

2. 設定出租條件

設定每日 HK$ 租金、取車地點、可出租日期與是否支援長租。

3. 接單與確認

收到 Renter 預訂後可在 Dashboard 接受或拒絕，系統自動檢查車輛可用性。

4. 交收與還車

按系統記錄的時間與地點交收，還車後查看每月收入統計與評價。

為什麼是 P2P，而不是傳統租車？

我們希望讓有需要短期用車的香港人，用最熟悉的方式找到車；同時讓車主把閒置車位變成穩定收入來源。

對 Renter · 彈性、透明、本地化

無需長約、無隱藏費用，所有金額以 HK$ 顯示，保險與條款以簡單中文解釋。

  • 支援常見電子支付（FPS、Apple Pay、AlipayHK 等）
  • Dashboard 顯示進行中、即將到期與歷史訂單
  • 清楚列出車輛里程、變速箱、油耗與特色配備

對 Owner · 安全、穩定、可控

完整的租單與收入報表，配合平台審核與保險機制，讓你安心把車放上平台。

  • 可自由設定每日 HK$ 價格與可出租時段
  • Owner Dashboard 顯示所有車輛與狀態（available / rented / maintenance）
  • 系統支援延長租期與額外付款流程（需 Owner 同意）
